## What I Build

- End-to-end automation with Playwright, Cypress, Selenium, TypeScript, Java and Python.
- BDD scenarios with Cucumber focused on readable business flows.
- API tests and contract checks with Rest Assured and Postman-style validation.
- SQL and backend validation notes for data quality.
- Test plans, test cases, regression coverage, smoke checks and exploratory testing.
- Evidence-first reports with screenshots, traces, logs and defect context.
- Public-safe manual QA evidence: expected result, actual result, impact, retest notes and risk.
- AI-assisted QA documentation and short AI animation experiments with original character work.
